# Subagent Monitor Assets

These assets define deterministic subagent monitoring reports.

- `manifest.json`: asset inventory and usage map.
- `subagent-monitor-report-contract.json`: required report fields.
- `timeout-policy.json`: bounded timeout and non-wall-clock evidence rules.
- `typed-result-policy.json`: one typed result per agent.
- `aggregation-policy.json`: blocker, warning, and partial-failure aggregation rules.
- `evidence-policy.json`: evidence tag and validation check requirements.
