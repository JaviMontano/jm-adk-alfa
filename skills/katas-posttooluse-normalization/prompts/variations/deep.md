# Deep Variation

Úsala cuando hay varias familias de payloads, múltiples schemas o migración gradual.

Incluye:

- Inventario de tools y matchers.
- Matriz de payload crudo a JSON canónico.
- Fallbacks por campo.
- Revisión de `additionalContext`.
- Riesgos de dialectos nuevos y estrategia de fixtures.
