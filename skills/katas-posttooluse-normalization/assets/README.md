# Assets

Deterministic policy assets for `katas-posttooluse-normalization`.

- `posttooluse-normalization-contract.json`: required report fields.
- `updated-output-policy.json`: `updatedMCPToolOutput` shape and raw payload exclusion.
- `normalization-policy.json`: schema, status map, fallback, and matcher rules.
- `evidence-policy.json`: accepted evidence types.
