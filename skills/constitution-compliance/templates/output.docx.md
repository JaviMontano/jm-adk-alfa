# Constitution Compliance DOCX Template

Use this Markdown structure when a Constitution compliance report must be
converted to DOCX. [EXPLICIT]

## Document Metadata

- Title: Constitution Compliance Report - `{artifact}`
- Author: JM Labs
- Constitution version: v6.0.0
- Gate: `{G0|G1|G2|G3|multi-gate}`

## Required Sections

1. Summary table.
2. 18-principle matrix.
3. G0-G3 gate impact.
4. Violations.
5. Missing evidence.
6. Remediation plan.
7. Decision.
8. Caveats.

## Formatting Rules

- Use system fonts only.
- Preserve evidence tags in all factual claims.
- Do not omit empty sections; use `none` with tagged evidence when applicable.
- Never mark the document ready when the Markdown source is blocked or
  `not_verified`.
