# Katas Tool Description Quality Assets

These assets define the deterministic contract for tool description quality reports.

- `tool-description-contract.json`: required report fields and Guardian decisions.
- `description-quality-policy.json`: required description elements and blocked wording.
- `routing-risk-policy.json`: overlap, rename, split, and prompt-bias rules.
- `action-priority-policy.json`: top action count and sorting.
- `evidence-policy.json`: accepted evidence types and blocked claims.

All assets are static and offline.
