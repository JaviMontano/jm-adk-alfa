# Context Window Management — Meta Prompt

Activate this skill when the request mentions context window management, token
budgeting, context trimming, compression, compaction preparation, or preservation
priority under context pressure.

Do not activate for browser windows, operating-system windows, UI layout windows,
or generic summaries that do not require token budgeting.
