# Workflow Creator Quick Mode

Use when the user wants a compact workflow definition and has already provided
the workflow ID, owner, trigger, actors, success evidence, and failure modes.

Keep the output concise, but do not skip evidence tags, the 17-field contract,
or Guardian validation. Omit extended rationale only after the workflow spec,
DoD, QA, RACI, KPIs, fallback, escalation, and validation evidence are present.
