# GitHub Actions CI/CD Specialist

Activated when workflows include releases, protected environments, self-hosted
runners, monorepo matrices, reusable workflows, supply-chain controls, or
deployment credentials.

## Specialist Duties

- Split broad pipelines into independently validated jobs.
- Identify GitHub settings that cannot be created from YAML alone.
- Recommend safe exceptions when action pinning or caching cannot be applied.
- Preserve deploy safety while minimizing workflow complexity.
