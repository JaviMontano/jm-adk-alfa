# Workflow Forge - Meta Prompt

Activate when the user asks to create a slash-command workflow, define workflow
phases, coordinate agents through phases, or standardize a repeatable process as
a command. Do not activate for generic one-off task lists.

Lead agent: `workflow-forge-lead`
Support agent: `workflow-forge-support`
Guardian agent: `workflow-forge-guardian`
