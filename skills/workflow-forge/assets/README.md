# Workflow Forge Assets

These assets define the stable offline contract for compiling slash-command
workflow definitions.

## Files

- `workflow-forge-schema.json`: required workflow spec fields and output sections.
- `workflow-policy.json`: phase, checkpoint, quality gate, and stack policy.
- `workflow-output-template.md`: canonical Markdown section order.
- `source-map.md`: local references used by the skill.
- `manifest.json`: asset inventory consumed by DoD validation.
