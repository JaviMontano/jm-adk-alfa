# Assembly Skill — Quick Mode

Run Phase A diagnostic and Phase D report only.

Required output:

- X-Ray scorecard summary.
- Mode/result as `quick` and `DIAGNOSTIC` or `BLOCKED`.
- `No files modified`.
- Next step for standard/deep if repair is needed.

Never modify files or claim `CERTIFIED` in quick mode.
