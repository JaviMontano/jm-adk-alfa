# Quality Gatekeeper Assets

Deterministic assets used by `quality-gatekeeper`.

- `gate-criteria.json`: canonical G0-G3 sequence and required criteria.
- `report-contract.json`: required JSON report sections, statuses, severities,
  and decision rules.
- `evidence-policy.json`: evidence tag vocabulary and assumption warning
  threshold.
- `score-history-schema.json`: proposed score-history entry fields.
- `activation-policy.json`: activation and false-positive routing rules.
- `manifest.json`: asset inventory used by the DoD validator.
