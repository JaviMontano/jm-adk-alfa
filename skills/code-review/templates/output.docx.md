# DOCX Template - Code Review

## Structure

1. Title: "Code Review Report"
2. Scope table: review type, sources reviewed, missing inputs, depth
3. Findings table: ID, severity, category, file, line, claim, suggested action
4. Positive patterns
5. Validation evidence
6. Decision
7. Risks and limits

## Formatting

- Heading 1: Poppins-compatible fallback, 24 pt, navy
- Heading 2: Poppins-compatible fallback, 16 pt, blue
- Body: system sans-serif, 10-11 pt
- Findings severities: BLOCKER red, MAJOR orange, MINOR blue, NIT gray
- No generated current date unless caller supplies `review_date`
