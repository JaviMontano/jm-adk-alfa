# Prompt Forge - Quick Mode

Use quick mode only when the user needs a fast draft or review. Keep the same deterministic gate:

- State mode, source boundary, target platform, and output contract.
- Include the Playbook or scorecard essentials.
- Include at least one happy path, one edge case, and one adversarial test.
- Keep evidence concise, but do not drop source-boundary or no-invention checks.
