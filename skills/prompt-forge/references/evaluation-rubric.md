# Prompt Forge Evaluation Rubric

Score each criterion from 1 to 10. Scores below 8 require a repair.

| Criterion | Passing Standard |
|---|---|
| Foundation | Role, objective, and constraints are clear. |
| Accuracy | Claims and methods are source-backed. |
| Quality | Writing is professional and precise. |
| Density | Token use is efficient without losing controls. |
| Simplicity | Structure is easy to follow. |
| Clarity | Instructions are unambiguous. |
| Precision | Boundaries are enforceable. |
| Depth | Edge cases and failure modes are covered. |
| Coherence | Sections reinforce the same behavior. |
| Value | The resulting prompt materially improves outcomes. |
