# Output Contract Checklist

- Contract source is explicit and parseable.
- Artifact is present and readable.
- Output format matches the declared format.
- Markdown outputs contain every required section.
- JSON outputs contain every required field.
- Required evidence tags use the canonical vocabulary.
- File names match declared naming style.
- Pass verdict is impossible when mandatory violations exist.
- Repair suggestions are specific and do not mutate files automatically.
- Validation packet matches `templates/schema.json`.
