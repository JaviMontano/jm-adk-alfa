# Output Contract Enforcer - Deep Analysis

Use deep mode for multi-file deliverables, release gates, or repeated contract failures.

1. Load the contract, artifact, assets, and relevant templates.
2. Produce a full validation packet with every check row.
3. Include deterministic repair suggestions.
4. Keep the same evidence-tag and naming rules as quick mode.
