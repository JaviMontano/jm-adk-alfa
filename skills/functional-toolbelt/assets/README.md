# Functional Toolbelt Assets

These assets support deterministic use of the six functional analysis tools.

| Asset | Purpose |
|---|---|
| `toolbelt-tools.json` | Canonical six-tool registry. |
| `event-storming-card-template.md` | Event/command/hot-spot card format. |
| `story-map-lane-template.md` | Story map activity/release lane format. |
| `decision-table-template.md` | Business rule decision table format. |
| `gwt-scenario-template.md` | Given/When/Then scenario format. |
| `traceability-matrix-schema.json` | Coverage and required traceability fields. |
| `anti-pattern-rules.json` | Requirements quality anti-pattern categories. |

Use `scripts/compile-functional-toolbelt.py` to validate a six-tool input and produce a repeatable Markdown report.
