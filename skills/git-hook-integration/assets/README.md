# Assets for git-hook-integration

[CODE] These assets define deterministic Git hook integration contracts used by `scripts/compile-git-hook-integration.py`.

## Files

- `git-hook-integration-schema.json` defines required input fields and allowed hook stages/managers.
- `hook-stage-model.json` defines canonical Git hook stages and safety expectations.
- `conventional-commit-policy.json` defines commit types, header limits, and message examples.
- `install-strategy-model.json` defines supported installation strategies and guardrails.
- `validation-command-catalog.json` defines deterministic validation command categories.
- `git-hook-integration-template.md` defines the Markdown output skeleton.
