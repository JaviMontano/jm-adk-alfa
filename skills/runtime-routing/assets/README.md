# Runtime Routing Assets

Deterministic assets for runtime route decisions.

- `runtime-routing-contract.json`: required report fields and Guardian decisions.
- `runtime-catalog-policy.json`: allowed runtime ids and permission levels.
- `evidence-policy.json`: allowed evidence types and required evidence fields.
- `capability-matrix-policy.json`: required matrix fields and validation statuses.
- `fallback-policy.json`: fallback and missing-data handling requirements.

The offline validator consumes these assets directly.
