# AI Content Detection - Deep Mode

Use deep mode for policy, compliance, education, publication, or disputed provenance.

Additional requirements:
- separate detector output from stylometric and metadata evidence
- include contradictory signals and confidence impact
- inspect watermark/provenance status explicitly
- require human review for any high-stakes action
- recommend disclosure or review process rather than punishment
