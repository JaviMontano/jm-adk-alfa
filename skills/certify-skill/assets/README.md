# Certify Skill Assets

Deterministic assets used by `certify-skill`.

- `certification-phases.json`: phase/check inventory for S, F, B, W, C, M, and
  rubric dimensions.
- `certification-level-policy.json`: exact formulas for MOAT, CERTIFIED,
  CONDITIONAL, and BLOCKED.
- `report-contract.json`: required JSON report sections and fields.
- `evidence-policy.json`: accepted evidence tags and evidence requirements.
- `activation-policy.json`: activation and false-positive routing rules.
- `manifest.json`: asset inventory used by the DoD validator.
