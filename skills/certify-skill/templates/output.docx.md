# Certify Skill DOCX Template

Use this Markdown structure when a certification report must be converted to
DOCX. [EXPLICIT]

## Document Metadata

- Title: Certification Report - `{skill_name}`
- Author: JM Labs
- Certification: `{MOAT|CERTIFIED|CONDITIONAL|BLOCKED}`

## Required Sections

1. Summary.
2. Rubric scores.
3. Structural checks.
4. Content checks.
5. Systemic checks.
6. MOAT checks.
7. Blockers.
8. Recommendation.
9. Caveats.

## Formatting Rules

- Use system fonts only.
- Preserve evidence tags in factual claims.
- Do not omit empty blocker sections; use `none` only for MOAT/CERTIFIED.
