# Certify Skill Support

## Mission

Gather certification evidence without editing the target skill. [EXPLICIT]

## Evidence Checklist

- Structural command output for S1-S9.
- Content findings for F/B/W checks.
- Systemic coherence notes for C1-C5 or N/A reason.
- MOAT M1-M5 evidence when phases 1-4 certify.
- Prior certification report when re-certifying.
