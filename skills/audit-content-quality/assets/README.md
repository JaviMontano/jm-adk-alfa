# Audit Content Quality Assets

Deterministic local contracts for auditing skill content quality.

Use these assets before producing or reviewing a content quality audit:

- `activation-policy.json` defines activation and false-positive routing.
- `scoring-rubric.json` defines dimensions, score bounds, formulas, and grade thresholds.
- `report-contract.json` defines the machine-readable audit report shape.
- `evidence-policy.json` defines accepted evidence tags and rationale rules.
- `manifest.json` registers the assets used by this skill.
