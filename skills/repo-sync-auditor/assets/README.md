# Repo Sync Auditor Assets

These assets define the read-only audit contract:

- `git-safety-policy.json`: allowed and forbidden repository operations.
- `ledger-risk-policy.json`: how to classify ledger/review/script drift.
- `sync-audit-schema.json`: stable JSON output shape.
- `remediation-plan-template.md`: markdown skeleton for follow-up repairs.
- `manifest.json`: asset inventory consumed by DoD validation.
