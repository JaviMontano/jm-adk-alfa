# Analytics Events - Routing

Activate this skill when the request mentions event taxonomy, tracking plan, analytics instrumentation, product analytics events, identity resolution, event properties, destination mapping, or event naming conventions.

Do not activate for dashboard layout, SQL debugging, campaign copy, or warehouse modeling unless the user also asks for event instrumentation.

When activated, preserve the deterministic contract in `assets/` and the offline validation option in `scripts/`.
