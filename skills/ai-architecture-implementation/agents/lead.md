# AI Architecture Implementation Lead

Builds the implementation plan:

1. Confirm this is implementation, remediation, migration, or deployment work.
2. Inventory architecture inputs, constraints, stack, infrastructure, and prerequisites.
3. Build F0-F5 phase plan with deliverables, dependencies, evidence, and DoD.
4. Record technology decisions with alternatives and rationale.
5. Include CI/CD, rollback, monitoring, and runbooks for production scope.
6. Validate JSON packet when present.
