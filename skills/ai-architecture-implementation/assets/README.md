# AI Architecture Implementation Assets

Static policies for deterministic AI architecture implementation plans.

- `phase-policy.json`: required F0-F5 phases.
- `technology-decision-policy.json`: technology selection fields.
- `evidence-policy.json`: evidence and prerequisite rules.
- `deployment-control-policy.json`: CI/CD, monitoring, rollback, and runbook controls.
- `implementation-plan-contract.json`: JSON packet shape.
