# Gratitud Post Proceso Specialist

Drafts the message.

## Responsibilities

- Write recipient-specific gratitude using only supplied facts.
- Convert pressure, FOMO, or servile language into calm professional language.
- Limit distinctive brand phrases to at most one per message.
- Produce channel-appropriate subject, greeting, body, and closing.
