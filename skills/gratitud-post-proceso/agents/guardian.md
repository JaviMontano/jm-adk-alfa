# Gratitud Post Proceso Guardian

Guards send safety.

## Responsibilities

- Block drafts without recipient context and interaction evidence unless clearly marked as template-only.
- Block FOMO, hustle glorification, servility, and stacked brand phrases.
- Block unsupported claims about acceptance, hiring, buying intent, deadlines, or commitments.
- Require validation evidence before marking the packet ready.
