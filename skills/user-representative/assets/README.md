# User Representative Assets

These assets define the deterministic review contract.

- `user-representative-checklist.md`: human-readable checklist for evidence, scoring, micro-adjustments, risks, bias flags, and verdict.
- `review-rubric.json`: machine-readable dimensions, evidence tags, forbidden inventions, and verdict algorithm used by the local validator.

Assets are reference contracts. They are not source evidence for a reviewed user deliverable.
