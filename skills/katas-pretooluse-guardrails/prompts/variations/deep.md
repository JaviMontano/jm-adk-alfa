# Deep Variation

Úsala cuando hay varias tools con side-effects, políticas compuestas o recarga desde JSON.

Incluye:

- Matriz de reglas por tool.
- Cobertura de `allow`, `deny` y `ask` cuando aplique.
- Evidencia de recarga de política.
- Prueba de prompt injection bloqueado.
- Revisión de matchers que puedan dejar rutas sin hook.
