# Assets

Deterministic policy assets for `katas-pretooluse-guardrails`.

- `pretooluse-guardrail-contract.json`: required report fields.
- `permission-decision-policy.json`: allowed decisions and denial requirements.
- `policy-source-policy.json`: accepted policy sources and reload rules.
- `side-effect-policy.json`: evidence required for pre-execution blocking.
- `evidence-policy.json`: accepted evidence types.
