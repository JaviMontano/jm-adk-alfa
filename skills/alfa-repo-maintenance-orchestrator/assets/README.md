# Alfa Repo Maintenance Orchestrator Assets

Deterministic policy assets for JM-ADK Alfa maintenance orchestration.

- `orchestration-report-contract.json` defines the required report schema.
- `write-blocker-policy.json` defines write blockers for mutating phases.
- `integration-matrix.json` lists required coordinating skills.
- `validation-gates.json` lists minimum validation gates.
- `manifest.json` records this asset bundle.
