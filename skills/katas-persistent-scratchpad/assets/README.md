# Katas Persistent Scratchpad Assets

Deterministic contracts for Kata 18 persistent scratchpad reports. These assets define the allowed file path, fixed sections, evidence rules, read-once access, append-only writes, and JSON report shape used by the local validator.

Use `manifest.json` as the index and `report-contract.json` with `scripts/validate_kata_scratchpad_report.py` for fixture validation.
