# adaptive-investigation-method assets

This directory contains the deterministic contract for adaptive investigation loop specifications.

## Files

- `manifest.json` lists every asset and its consumer.
- `investigation-schema.json` defines required fields for goals, budgets, maps, hypotheses, deep-dives, replans, stop conditions, and deliverables.
- `investigation-policy.json` defines allowed tools, budget rules, valid replan triggers, stop reasons, and anti-pattern tokens.
- `report-template.md` documents the Markdown report shape generated by the compiler.

The script in `../scripts/compile-adaptive-investigation.py` loads these assets and fails closed when an investigation loop is unbounded, non-traceable, or prone to reflexive replanning.
