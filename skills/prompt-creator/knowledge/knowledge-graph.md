# Prompt Creator - Knowledge Graph

## Core Concepts

- prompt-creator: deterministic prompt artifact generator
- prompt-type-matrix: canonical type, redirect, and required-section policy
- prompt-contract-checklist: reusable frontmatter, body, placeholder, and decision gate
- source-agent: inspected agent constitution or explicit missing-source gap
- prompt-artifact-validator: script-backed check for generated markdown prompts
- downstream-boundary: generated prompt does not execute the downstream task

## Dependencies

- Upstream: auto-prompt-matching, input-analyst, agent-constitution-creator
- Downstream: workflow-orchestration, output-contract-enforcer, quality-gatekeeper
