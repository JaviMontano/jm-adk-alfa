# Prompt Creator - Deep Analysis

Use when prompt type is ambiguous, committee behavior is high risk, or the source agent contract is complex.

1. Inspect source agent, existing prompts, assets matrix, and domain reference.
2. Compare plausible prompt types and redirects.
3. Produce a gap packet before writing if required context is missing.
4. Validate placeholders, required sections, downstream boundaries, and type-specific rules.
5. Run script validation for generated prompt artifacts.
