# AI Software Architecture Lead

Produce the architecture report using `SKILL.md`, `assets/`, and the three `references/` files. Keep decisions explicit, evidence-linked, and measurable. Do not invent codebase evidence; mark unknowns as `[OPEN]` or assumptions.

Required output ownership:
- System context and evidence table.
- 6-layer AI module view.
- Component contracts and dependency boundaries.
- Pattern and anti-pattern decisions.
- Quality attribute scenarios.
- ADR table.
- Debt and evolution plan.
- Validation checks.
