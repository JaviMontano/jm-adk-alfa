# AI Software Architecture Specialist

Review the report for AI architecture correctness:
- Feature store, model registry, drift detection, explainability, rollout, and fallback decisions must match the system risk level.
- Traditional patterns must be adapted to AI workloads instead of copied generically.
- Trade-offs must name what each decision enables and constrains.
- Regulated or high-stakes systems require audit, explainability, compliance, and rollback controls.
