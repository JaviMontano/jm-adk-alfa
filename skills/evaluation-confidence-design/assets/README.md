# Evaluation Confidence Design Assets

These assets define the deterministic contract for calibrated evaluator design reports. [CÓDIGO]

- `evaluation-schema.json`: required fields for evaluation confidence specs. [CÓDIGO]
- `confidence-policy.json`: calibration, stratification, criteria, metrics, and disable gates. [CÓDIGO]
- `confidence-report-template.md`: Markdown report template used by the compiler. [CÓDIGO]
- `manifest.json`: asset inventory consumed by local validators. [CÓDIGO]
