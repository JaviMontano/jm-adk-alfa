# Analytics Implementation Assets

These static JSON assets define the deterministic contract for GA4/Firebase Analytics implementation outputs.

## Files

- `manifest.json`: DoD asset manifest.
- `analytics-implementation-contract.json`: Required report sections and validation checks.
- `ga4-policy.json`: Setup and stream requirements.
- `event-policy.json`: Event parameter and naming requirements.
- `conversion-policy.json`: Conversion marker requirements.
- `bigquery-policy.json`: Export, retention, partitioning, and PII handling requirements.
- `dashboard-policy.json`: Dashboard data source and metric requirements.
- `evidence-policy.json`: Evidence tags and provenance requirements.
