# Knowledge Management Assets

These assets define the deterministic contract for knowledge capture,
searchability, decay prevention, and offline validation.

- `manifest.json` lists every asset and its consumer.
- `evidence-policy.json` defines allowed evidence tags.
- `knowledge-taxonomy.json` defines item, gap, and status values.
- `searchability-policy.json` defines retrieval metadata rules.
- `freshness-policy.json` defines decay windows based on explicit dates.
- `report-contract.json` defines JSON report fields validated by scripts.
