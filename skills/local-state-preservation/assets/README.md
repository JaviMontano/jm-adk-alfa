# Assets

These assets define the preservation packet contract used by `local-state-preservation`.

Use them as machine-checkable policy inputs:

- `preservation-report-contract.json`: required top-level JSON report shape.
- `surface-inventory-policy.json`: required inventory surfaces and fields.
- `archive-manifest-policy.json`: artifact metadata policy.
- `private-path-policy.json`: private path exclusion rules.
- `non-touch-policy.json`: surfaces that default to inventory-only.
- `preservation-template.md`: report authoring template.
