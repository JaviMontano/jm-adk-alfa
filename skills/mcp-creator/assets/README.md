# MCP Creator Assets

These assets define deterministic MCP configuration planning.

- `manifest.json`: asset inventory and usage map.
- `mcp-config-plan-contract.json`: required plan fields.
- `transport-policy.json`: stdio/http requirements and blocked transports.
- `scope-policy.json`: local/project/user/plugin scope rules.
- `secret-policy.json`: env-var placeholder and hardcoded secret blockers.
- `preflight-policy.json`: existing config, collision, live validation, and rollback requirements.
- `evidence-policy.json`: evidence tag and validation check requirements.
