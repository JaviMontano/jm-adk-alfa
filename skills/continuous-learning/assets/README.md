# Continuous Learning Assets

These assets define the deterministic contract for extracting reusable project
learning from debates, discoveries, decisions, and incidents.

- `source-event-policy.json` defines valid source event and three-output fields.
- `insight-policy.json` defines insight domains, statuses, and required fields.
- `duplicate-policy.json` defines refinement and supersession behavior.
- `amendment-policy.json` defines recurrence thresholds for amendment proposals.
- `update-plan-policy.json` defines safe update paths and index requirements.
- `learning-report-contract.json` defines the machine-readable report shape.
