# Continuous Learning — Body of Knowledge

## Canon

Continuous learning converts debates and discoveries into reusable, routed
knowledge. It prevents repeated debate by consulting existing insights first,
then adding, refining, superseding, or blocking candidate insights.

## Quality Metrics
| Metric | Target | How to Measure |
|--------|--------|---------------|
| Prior search | 100% | Relevant insight domains searched before creating entries |
| Three outputs | 100% | Answer, refinements, and gaps captured |
| Trigger quality | 100% | Each insight has actionable trigger conditions |
| Duplicate safety | 100% | No duplicate active insight created |
| Amendment threshold | 100% | Recurrence count >= 3 for proposed amendments |

## Insight Status

- active: reusable pattern is accepted as default.
- tentative: useful but evidence is incomplete or early.
- superseded: replaced by a newer insight.
- rejected: not reusable or duplicate without refinement.
