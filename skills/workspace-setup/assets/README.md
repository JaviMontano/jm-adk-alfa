# Workspace Setup Assets

These assets define the deterministic contract for local workspace profile setup plans.

- `manifest.json`: asset inventory and usage map.
- `workspace-setup-plan-contract.json`: required setup plan fields.
- `runtime-preferences-policy.json`: profile preference requirements.
- `command-policy.json`: allowed, prohibited, and escalation-required command policy.
- `privacy-policy.json`: local-only and no-secret requirements.
- `write-policy.json`: dry-run, apply, overwrite, and gitignore requirements.
- `evidence-policy.json`: evidence tag and validation requirements.
