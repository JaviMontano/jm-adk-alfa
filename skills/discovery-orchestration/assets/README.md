# Discovery Orchestration Assets

These assets define deterministic pipeline sequencing, gate enforcement,
deliverable tracking, dependency validation, and offline orchestration-packet
checks.

- `manifest.json` lists every asset and consumer.
- `pipeline-schema.json` defines pipeline and phase fields.
- `gate-policy.json` defines quality gate requirements.
- `deliverable-policy.json` defines deliverable states.
- `dependency-policy.json` defines dependency graph rules.
- `status-taxonomy.json` defines canonical status values.
- `report-contract.json` defines the orchestration packet contract.
