# Custom Tooling Extension Assets

These assets define the deterministic contract for Claude Code custom tooling decisions. [CÓDIGO]

- `extension-schema.json`: required fields for command/skill extension specs. [CÓDIGO]
- `extension-policy.json`: decision, scope, interface, and safety gates. [CÓDIGO]
- `extension-report-template.md`: report template used by the compiler. [CÓDIGO]
- `manifest.json`: asset inventory consumed by local validators. [CÓDIGO]
