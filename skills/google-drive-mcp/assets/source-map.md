# Google Drive MCP Source Map

## Local MCP Sources

- [DOC] `docs/google-workspace-mcp-setup.md` defines the local `workspace-mcp`
  setup, `extended` tool tier, Drive service availability, read-only mode, and
  service permission levels.
- [DOC] `docs/mcp-integration.md` defines JM-ADK MCP placement, project-level
  `.mcp.json`, `workspace-mcp` as the Google Workspace server, and MCP security
  rules such as environment variables for secrets.
