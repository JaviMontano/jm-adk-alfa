# DOCX Template - Code Review Checklist

## Document Structure

1. Title: "Code Review Checklist Report"
2. Scope table
3. Domain score table
4. Checklist results table
5. Findings table
6. Missing evidence
7. Validation
8. Decision
9. Risks and limits

## Formatting

- Heading 1: system sans-serif, 24 pt, navy
- Heading 2: system sans-serif, 16 pt, blue
- Blocking failures: red
- Non-blocking failures: orange
- Pass: blue
- No generated current date unless caller supplies `review_date`
