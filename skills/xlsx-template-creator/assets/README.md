# xlsx-template-creator assets

This directory contains the deterministic contract for XLSX template specifications.

## Files

- `manifest.json` lists every asset and its consumer.
- `xlsx-template-schema.json` defines required workbook, sheet, column, named range, and validation fields.
- `template-policy.json` defines required sheets by template type and formatting rules.
- `formula-policy.json` defines safe formula and data-validation constraints.
- `report-template.md` documents the Markdown report shape generated by the compiler.

The script in `../scripts/compile-xlsx-template.py` loads these assets and fails closed when the spec is not render-safe.
