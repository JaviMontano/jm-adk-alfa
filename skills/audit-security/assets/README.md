# Audit Security Assets

Deterministic local contracts for static plugin security audits.

- `activation-policy.json` defines activation, false-positive routing, and clarification rules.
- `scan-policy.json` defines the six scan categories, severity taxonomy, and placeholder policy.
- `report-contract.json` defines the JSON report shape.
- `evidence-policy.json` defines accepted evidence and remediation requirements.
- `manifest.json` registers the assets used by this skill.
