# Workspace Governance Assets

These assets define deterministic governance for the gitignored `workspace/`
interaction layer.

- `gitignore-policy.json` defines ignore requirements.
- `session-policy.json` defines dated session folder rules.
- `task-bridge-policy.json` defines tasklog bridge mapping.
- `directory-readme-policy.json` defines README coverage.
- `stale-review-policy.json` defines stale session handling.
- `workspace-governance-report-contract.json` defines the report shape.
