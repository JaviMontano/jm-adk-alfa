# User Prompt Filter Assets

These assets define the deterministic prompt filtering contract.

| Asset | Purpose |
|---|---|
| `filter-input-schema.json` | Required input fields for prompt filtering |
| `threat-taxonomy.json` | Threat classes, evidence patterns, and base scores |
| `risk-scoring-policy.json` | Decision thresholds and severity mapping |
| `sanitization-policy.json` | Safe transformation and redaction rules |
| `output-schema.json` | JSON output contract |
| `source-map.md` | Traceability between assets, scripts, and docs |
