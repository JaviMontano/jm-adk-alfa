# Task Engine — Deep Mode

Use deep mode for high-stakes, ambiguous, multi-domain, or explicitly confidence-calibrated reasoning.

Deep mode adds:

- complete DSVSR packet,
- confidence rationale for every sub-problem,
- explicit verification across LOGIC, FACTS, COMPLETENESS, and BIAS,
- reflection retry decision,
- and non-executive wording when global confidence is below target.

Load only references needed for the detected gap; do not bulk-load unrelated files.
