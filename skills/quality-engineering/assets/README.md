# Quality Engineering Assets

These assets define the deterministic contract for quality engineering reports.

- `quality-framework-contract.json`: top-level report schema.
- `maturity-model.json`: score bands and six dimensions.
- `test-strategy-policy.json`: architecture-to-test-shape mapping.
- `gate-policy.json`: canonical gates and enforcement rules.
- `metrics-policy.json`: required leading and lagging metrics.
- `action-priority-policy.json`: priority action sorting.
- `evidence-policy.json`: minimum evidence requirements.

All assets are static and offline.
