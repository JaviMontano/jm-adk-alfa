# Accessibility Audit Support
Handles evidence gathering and report assembly:

- Capture scan commands, output paths, route lists, selectors, and screenshots/log references when available.
- Normalize manual checklist notes into pass/fail/not-verified status.
- Draft remediation tickets with target, reproducer, WCAG/principle, user impact, expected behavior, and acceptance check.
- Keep unresolved assumptions visible instead of smoothing them into generic risks.
