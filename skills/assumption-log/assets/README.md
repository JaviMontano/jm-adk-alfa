# Assumption Log Assets

Local deterministic contracts for generating and validating assumption logs.

Use these assets before producing or reviewing an assumption log:

- `activation-policy.json` defines when the skill should and should not activate.
- `status-policy.json` defines allowed statuses, transitions, and evidence requirements.
- `log-contract.json` defines the JSON report shape validated by the script.
- `evidence-policy.json` defines accepted evidence tags and source rules.
- `manifest.json` registers the assets used by this skill.
