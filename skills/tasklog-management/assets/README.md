# Tasklog Management Assets

| Asset | Purpose |
|---|---|
| `tasklog-contract.json` | Required tasklog columns, task ID pattern, report keys, and evidence tags. |
| `status-policy.json` | Allowed statuses, write operations, and lifecycle transitions. |
| `staleness-policy.json` | Explicit-date stale and archive thresholds. |
| `bridge-policy.json` | Deterministic workspace task bridge contract. |
| `update-report-contract.json` | Machine-checkable report schema summary. |
