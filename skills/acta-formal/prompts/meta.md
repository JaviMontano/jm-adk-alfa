# Acta Formal Meta Prompt

Review whether `acta-formal` should activate, whether the scope is safe, and which support agents should participate.

## Activation Check

- Trigger match
- Domain fit
- Sufficient input
- No safer specialized skill available

## Review Questions

1. Is the request formal enough, or should it route to `meeting-notes`?
2. Are quorum threshold/source and attendance enough to validate agreements?
3. Are folio, president, secretary, owners, and deadlines supplied or marked `por_confirmar`?
4. Does the output preserve sections I-VIII in Markdown and HTML?
5. Are Drive/Gmail actions blocked until explicit confirmation?
6. Did any agent invent a formal fact to make the acta look complete?
