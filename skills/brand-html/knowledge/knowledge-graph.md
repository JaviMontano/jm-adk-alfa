# Brand HTML - Knowledge Graph

## Core Concepts

- [[single-file-html]] - One self-contained HTML document with inline CSS.
- [[brand-token-css]] - CSS variables generated from brand or fallback tokens.
- [[svg-favicon]] - Browser favicon link using `type="image/svg+xml"`, with no
  remote or base64 dependency.
- [[dependency-boundary]] - No external JavaScript, base64 images, or remote
  assets unless explicitly allowed.
- [[semantic-landmarks]] - `header`, `nav`, `main`, `section`, and `footer`.
- [[responsive-css]] - Media queries and stable responsive constraints.
- [[false-positive-routing]] - DOCX/PDF/XLSX/slides/token-only requests route
  away.
