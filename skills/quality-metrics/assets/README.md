# Quality Metrics Assets

These assets define the deterministic contract for quality metrics reports.

- `quality-metrics-contract.json`: required report fields and decisions.
- `metrics-thresholds.json`: canonical metrics, thresholds, warning bands, and score points.
- `gate-policy.json`: required gate enforcement fields.
- `trend-policy.json`: deterministic trend regression rules.
- `action-priority-policy.json`: priority action sorting.
- `evidence-policy.json`: allowed evidence types and blocked claims.

All assets are static and offline.
