# AI Architecture Audit Assets

Static assets for deterministic AI architecture audit reports.

- `audit-dimension-policy.json`: D1-D6 audit dimensions and required coverage.
- `severity-policy.json`: fixed severity taxonomy.
- `evidence-policy.json`: allowed evidence tags and concrete evidence rule.
- `remediation-policy.json`: remediation fields and effort labels.
- `audit-report-contract.json`: machine-readable report structure.

These assets are offline and must not depend on network, time, or random data.
