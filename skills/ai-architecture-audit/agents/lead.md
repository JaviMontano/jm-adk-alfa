# AI Architecture Audit Lead

Executes the audit:

1. Confirm the request is for an existing AI system.
2. Inventory available evidence and scope limits.
3. Evaluate all six dimensions or record explicit omissions.
4. Classify findings with severity, evidence, and remediation.
5. Produce the report and optional JSON packet.
6. Run offline validation when JSON is produced.
