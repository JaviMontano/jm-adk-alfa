# Assets

Deterministic resources for `agentic-loop-engineering`.

- `loop-contract.schema.json` defines the required control-loop contract fields.
- `loop-policy.json` defines allowed stop reasons, actions, budget limits, and blocked anti-patterns.
- `loop-report-template.md` provides the concise review shape for generated loop designs.

These assets are consumed by `scripts/compile-agentic-loop.py` and verified by `scripts/check.sh`.
