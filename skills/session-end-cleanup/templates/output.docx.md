# DOCX Template - Session End Cleanup

## Document Properties

- Title: Session End Cleanup Report
- Author: JM Labs
- Subject: Evidence-tagged closeout and next-session handoff

## Required Sections

1. Session Summary
2. Changes Completed
3. Decisions And Assumptions
4. Open Tasks
5. Insights Captured
6. Risks And Blockers
7. Validation Evidence
8. Durable Updates
9. Next Handoff
10. Guardian Decision

## Formatting

- Use built-in document fonts; do not fetch remote fonts.
- Render command names in monospace.
- Keep validation evidence as a table with command, status, and evidence.
