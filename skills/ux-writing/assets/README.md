# UX Writing Assets

These assets define the deterministic UX writing contract.

- `ux-writing-checklist.md`: human-readable checklist for findings, rewrites, accessibility, readability, and no-invention validation.
- `microcopy-patterns.json`: machine-readable patterns and blocked anti-patterns.
- `readability-rubric.json`: reading-level, sentence, paragraph, jargon, and accessibility targets.

Assets are contract references. They are not source facts for the user deliverable.
