# Google Workspace APIs Assets

These assets make the skill deterministic:

- `google-workspace-apis-schema.json` defines compiler input shape.
- `workspace-service-matrix.json` lists supported services and operations.
- `auth-scope-policy.json` defines least-privilege scope profiles.
- `mcp-tool-contract.json` maps MCP tools to service operations.
- `operation-policy.json` defines validation layers, secrets, and mutation gates.
- `google-workspace-apis-template.md` is the Markdown report template.
- `source-map.md` records official Google and MCP references.
