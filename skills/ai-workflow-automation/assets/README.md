# AI Workflow Automation Assets

These assets define deterministic workflow planning for AI steps, human approval
gates, handoffs, fallbacks, and offline validation.

- `manifest.json` lists every asset and consumer.
- `workflow-schema.json` defines workflow plan fields.
- `actor-taxonomy.json` defines allowed actor types.
- `approval-gate-policy.json` defines gate rules.
- `handoff-policy.json` defines handoff packet requirements.
- `failure-policy.json` defines bounded retry and fallback rules.
- `report-contract.json` defines the JSON plan contract.
