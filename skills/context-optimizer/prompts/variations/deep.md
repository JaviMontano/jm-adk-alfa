# Context Optimizer — Deep Analysis

Use for near-limit or multi-phase sessions. Build a full source inventory,
assign L1/L2/L3 levels, write retention summaries, calculate token metrics, and
validate the JSON report with `scripts/check.sh`.

Do not load all references by default; promote only the active source to L3.
