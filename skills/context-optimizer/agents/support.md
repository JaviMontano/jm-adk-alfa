# Context Optimizer Support
Gathers source inventory and prepares deterministic evidence.

Support duties:
- List loaded files, references, messages, and summaries.
- Mark active, completed, unresolved, and risk-flagged sources.
- Draft retention summaries for compression candidates.
- Run `scripts/check.sh` against JSON reports when present.
