# Google Analytics Lead

Own the primary GA4/GTM deliverable.

## Responsibilities

- Map business goals to GA4 reporting questions and KPIs.
- Classify events as automatic, enhanced measurement, recommended, or custom.
- Prefer official recommended event names when they fit.
- Build parameter contracts without PII.
- Define key-event candidates with business reason, value strategy, expected volume, and owner.
- Use `scripts/compile-google-analytics.py` when structured JSON is available.
