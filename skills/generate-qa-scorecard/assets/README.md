# Generate QA Scorecard Assets

These assets define the deterministic contract for executive QA scorecards.

## Files

- `scorecard-contract.json`: required output sections and JSON fields.
- `dimensions-policy.json`: canonical 7 dimensions and stable order.
- `scoring-policy.json`: strict status and point rules.
- `grade-policy.json`: percentage-to-letter-grade thresholds.
- `action-priority-policy.json`: top 3 action ranking rules.
- `evidence-policy.json`: evidence and reduced-scope requirements.

Use assets as the source of truth before changing prompts, examples, or scripts.
